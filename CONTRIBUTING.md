# Contributing

Thanks for your interest in ReqRun.

## Best ways to contribute

- improve docs clarity
- improve examples
- report confusing integration steps
- suggest additional starter repos for real-world stacks

## Before opening changes

Please keep the public surface focused on the actual product boundary:

- reliable OpenAI-compatible request execution
- small examples
- clear docs

Avoid adding:

- provider marketplace abstractions
- workflow platform features
- analytics product concepts that do not belong to ReqRun

## Issues

Use GitHub issues for:

- docs problems
- example bugs
- public SDK ergonomics
- missing integration guides

For sensitive security issues, use:

- security@reqrun.com
